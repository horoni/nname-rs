# nname-rs
rewrite of [nname](https://github.com/oldteamhost/nname) in blazingly fast :rocket: :rocket: :rocket: and memory-safe Rust language.
```
cargo build -r
```

ex:
```
./nname pidor -o -p 0x
0x7069646f72

./nname pidor -P h
70h 69h 64h 6fh 72h

./nname pidor -b -o -p 0b
0b0111000001101001011001000110111101110010

./nname pidor -O -o -p 0
0160151144157162
```
